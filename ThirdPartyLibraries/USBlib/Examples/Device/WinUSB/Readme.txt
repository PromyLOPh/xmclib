WINUSB demonstration application
================================

Overview
--------
This gives a simple reference application for implementing a WINUSB device.

Supported Platforms
-------------------
Infineon XMC4500 Application Kit
http://www.infineon.com/cms/en/product/evaluation-boards/KIT_XMC45_EE1_002/productType.html?productType=db3a30443cd75eda013cd984ee38047d#ispnTab1

Hardware Requirements
---------------------
Standard Human Machine Interface Satellite Card
http://www.infineon.com/cms/en/product/evaluation-boards/KIT_XMC4X_HMI_OLED_001/productType.html?productType=db3a304433b8a4100133daf9d7371127

Software Requirements
---------------------
Before running this demo for the first time on a new computer, you will need to supply the driver for the Host side.
When running under Windows, please install the Signed Driver (.INF file) located in WinUSB_Signed_Package_Driver folder. 

Getting Started
---------------
Download the program to the target board using one of the provided project files for the supported IDE.

Either press the reset button on your board or launch the debugger in your IDE to begin running the demo.

Run the WinUSB-Example.exe inside the HostTestApp folder in the Host to interact with the microcontroller.
