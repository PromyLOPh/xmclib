Description of example project
==============================
This example project depicts usage of FLASH driver.

Hardware Setup
===============
XMC1300 Bootkit 

How to test the application
============================
This example uses the XMC1_flash Low Level Driver to perform flash erase, 
program and verify operation. The correctness of the different operations 
can be monitored using the LEDs of the bootkit:
 - A blinking LED means the flash step found an error.
 - A lighting LED means the flash step is successful.
