CAN demo example

Description of example project
==============================
Communication in Loopback mode between two CAN Nodes. 
The frequency source used by the module is the High Precision External Oscillator (OSC_HP)
The user will see the onboard LED1(P4.0) once the content of the MO4(NODE 2) has been received and read in MO2 (NODE 1)

Hardware Setup
===============
XMC1400 Boot Kit
